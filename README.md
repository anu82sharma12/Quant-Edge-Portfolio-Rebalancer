#  Quant-Edge Portfolio Rebalancer

A reinforcement learning simulator for **dynamic asset allocation**, built with TensorFlow and OpenAI Gym.

###  Key Achievements
- Achieved **~15 % Sharpe ratio improvement** on synthetic portfolio
- Demonstrated **AI-driven portfolio optimization** via policy gradient RL
- Simulated **risk-adjusted rebalancing** with live market data
